# Personal Website

A personal website built with Eleventy and hosted on Netlify.

## Tech Stack

- **Static Site Generator**: Eleventy (11ty)
- **Templating**: Nunjucks
- **Styling**: CSS
- **Deployment**: Netlify

## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm start
```

### Build

Build the site for production:
```bash
npm run build
```

## Project Structure

- `src/` - Source files (templates, markdown content)
- `public/` - Generated static site (output directory)
- `src/_includes/` - Layout templates and partials
- `src/blog/` - Blog posts in markdown
- `src/books/` - Books tracking pages
